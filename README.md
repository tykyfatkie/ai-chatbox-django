## 🚀 Hướng dẫn khởi chạy (Getting Started)

Để chạy dự án này trên máy local của bạn, hãy mở 2 cửa sổ Terminal (hoặc Command Prompt) riêng biệt cho Backend và Frontend.

### 🐍 1. Backend (Django)

Mở terminal đầu tiên và thực thi các lệnh sau:

```bash
# Di chuyển vào thư mục chứa code backend
cd ai-chatbot-django

# Kích hoạt môi trường ảo (Virtual Environment)
.\venv\Scripts\activate

# Khởi chạy server
python manage.py runserver
```
